# Governance Guide

## Parameter Settings

| Parameter | Default | Purpose |
|-----------|---------|---------|
| max-slots-per-teacher | 100 | Prevents concentration |
| min-time-block | 1 | Validates time block inputs |

## Change Process

1. Contract owner proposes change
2. Call setter function with new value
3. Governance action counter increments
4. New parameter takes effect immediately
