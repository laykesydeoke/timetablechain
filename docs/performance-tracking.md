# Performance Tracking

Protocol performance metrics for TimeTableChain.

## Statistics

```clarity
(get-performance-stats)   ;; Comprehensive stats
(get-protocol-uptime)     ;; Blocks since deployment
```

## Fields

- `total-slots`: All-time slots created
- `total-transfers`: All-time transfers
- `total-tx`: Total transactions
- `uptime-blocks`: Blocks since deployment
